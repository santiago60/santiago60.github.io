Ngrok Con Python - Windows

Paso 1 : Nos Localizamos en la carpeta a compartir y ejecutamos : python -m http.server 80

Paso 2 : Nos ubicamos donde se encuentra el archivo 'ngrok.exe y ejecutamos : './ngrok.exe http 80

Ngrok Con Xampp - Windows

Paso 1 : Ejecutamos servidor Xampp

Paso 2 : Nos Localizamos en la carpeta htdocs y abrimos el archivo index.php

Paso 3 : modificamos el nombre 'ubicacion', header('Location: '.$uri.'/ubicacion');

CREAR ENTORNO VIRTUALENV

Paso 1 : Verificar Version: python --version

Paso 2 : instalar virtualenv: pip install virtualenv

Paso 3 : crear entorno: virtualenv -p 3.11.4 .venv

Paso 4 : Activarlo: source .venv/Scripts/activate

Paso 5 : Desactivarlo: deactivate

Ejecutar Servidor Tor .onion

          sudo apt update
          sudo apt install tor
          sudo nano /etc/tor/torrc

          Descomenta o agrega las siguientes líneas al final del archivo:
          HiddenServiceDir /var/lib/tor/oculto/
          HiddenServicePort 80 127.0.0.1:8080

          sudo systemctl restart tor

          cd /var/lib/tor/oculto/
          cat hostname

          sudo apt install python3
          python3 -m http.server 8080

          sudo apt install nginx
          sudo nano /etc/nginx/sites-available/oculto


          Agrega la siguiente configuración, reemplazando "8080" con tu puerto y ajustando las rutas según tus archivos:

          server {
              listen 127.0.0.1:8080;
              server_name tu_direccion_onion.onion;

              location / {
                  root /var/www/mi_servicio_oculto;
                  index index.html;
              }
          }

          Crea un enlace simbólico al directorio sites-enabled:
          sudo ln -s /etc/nginx/sites-available/oculto /etc/nginx/sites-enabled/

          sudo systemctl restart nginx
